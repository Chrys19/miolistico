# MiOlistico – Pacchetto pubblicazione (Netsons)

## Contenuto
- index.html
- robots.txt
- sitemap.xml
- assets/ (logo, og-image, foto)
- assets/icons/ (favicon e icone)

## Pubblicazione su Netsons
1) Carica tutti i file nella root del dominio (es. public_html/).
2) Verifica:
   - https://miolistico.it/
   - https://miolistico.it/robots.txt
   - https://miolistico.it/sitemap.xml

Nota: il form è in modalità mailto (apre la mail). Se vuoi invio reale, si integra Formspree/Zapier.
